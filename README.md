# Painel de Vendas

App mobile para vendedores acompanharem seus clientes, rupturas, impulso e take home.

## Como usar (gestor)

1. Toda semana: atualize o arquivo `Painel_de_Vendas.xlsx` com os dados da indústria
2. Acesse o link do app e faça upload do arquivo
3. Cada vendedor seleciona seu nome e vê o próprio painel

## Estrutura do arquivo Excel

O app lê a aba **PAINEL** do arquivo `Painel_de_Vendas.xlsx`.
Colunas necessárias:
- Código Cliente
- Vendedor
- Data Visita
- Nome Cliente
- Ruptura (tipo)
- Tipo Contrato
- Impulso Obj.
- Impulso Real.
- TH Obj.
- TH Real.
- Devedor Valor R$
- Picolé Campanha (Sim / Não)
